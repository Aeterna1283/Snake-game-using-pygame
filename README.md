# Snake-game-using-pygame
A simple implementation of snake using pygame, with graphics included.
<img width="721" height="644" alt="Screenshot 2026-02-21 162724" src="https://github.com/user-attachments/assets/a01f7e90-e565-4772-9e54-f329d747b5ef" />

Makes use of object oriented programmed, classes, and seperate functions. Simple game loop which updates every 60 seconds. 

The snake makes use of a vector for each node of its body. The apple spawns in random locations on the board.
